Este repositório contém scripts SQL para criação e manipulação de tabelas no banco de dados.

Estrutura dos Arquivos

Arquivo Descrição
01-Table_Cliente.sql Cria a tabela Cliente.
01-Table_Pedidos.sql Cria a tabela Pedidos.
01-populate.sql Insere dados iniciais nas tabelas.
02-Table-Cliente.sql Versão alternativa da tabela Cliente.

Como Usar

  1. Execute os scripts de criação das tabelas no seu banco de dados.
  2. Use os arquivos de inserção para popular os dados.
